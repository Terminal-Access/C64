
Set of ROM files is not included in source control,
but it is available at following URL:

http://kataklinger.com/wp-content/uploads/2014/06/c64_roms.zip

The archive should contain these files:

 * kernal.rom - content of ROM chip hosting OS
 * basic.rom - content of ROM chip hosting Basic interpreter
 * chargen.rom - Charachter ROM
 * d1541.rom - firmware of CBM 1541-II drive

Files should be copied to .\c64_roms folder of the project
